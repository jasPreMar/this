Task Icon Mode — Design Overview

After submitting from the floating panel, the panel shrinks to a persistent icon that tracks the hovered window

Lifecycle

State Flow
Floating Panel
User types + submits
→
Task Icon
Shrinks, sticks to window, pulses
→
Command Menu
On click or task completion

Icon States

Idle (not hovered)
← Pulsing opacity
0.4 ↔ 1.0
Hovered (expanded)
Running bash: git status...

Status Text Examples

Starting... when waiting
Thinking... when thinking block active
Running bash when tool is active
Reading file.swift when Read tool is active
Done when complete (briefly, before auto-opening command menu)

Window Tracking

At submission time, capture the CGWindowID of the window under the cursor and the icon's offset relative to that window's origin. A timer polls at ~0.25s to reposition the icon as the window moves. If the tracked window disappears (minimized/closed), the icon opens the command menu automatically.
